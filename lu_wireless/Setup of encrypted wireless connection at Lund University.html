Setup of encrypted wireless connection at Lund University

 På svenska
Almost all computers and most modern mobile phones can connect to the wireless network
at Lund university in an easy and safe (encrypted) way. We use the standard 802.1x with
protocols either TTLS+PAP or PEAP+MsChapv2 to authenticate using userid/password.
User certificates are not needed or used.

Connect to network eduroam using protocols in order:

WPA2-företag or WPA2-Enterprise if available, or else WPA2 if available, or else
WPA-företag or WPA-Enterprise if available, or else WPA-EAP or RSN or WPA.

Then select encryption AES or CCMP if available, or else TKIP.

Then select autentication method PEAP with protocol/method MsChapv2
or TTLS with protocol/method PAP.

Set Roaming-Id or Outer-Id or anonymous_identity as your username with @lu.se appended
(username is not your email address, username looks like inst-sig or inst12sig).

Do validate certificates: select GTE CyberTrust Global Root.
Linux users may have to fetch and install this root certificate.

Verify server name: For SecureW2 set .lu.se and for Intel ProSet set lu.se
and for Windows Zero Configuration set radius1.lu.se; radius2.lu.se
(other clients may have different syntax).

Fill in Username with @lu.se appended and Password, leave Domain empty.

You only have to do the above once, from then on you will be connected automatically.
